# xc-s/emz
* This is an open source tool written 100% in Python and is completely safe to use.
* Its function is to quickly copy emojis and stickers from one server to another.

# 〢 Tutorial
##  Requirements

* Python 3.x : https://www.python.org/downloads/
* Python libraries: `requests`, `pystyle`.
```bash
python -m pip install requests
```
```bash
python -m pip install pystyle
```

>[!NOTE]
> * **You can run the `install.bat` file to quickly install the required libraries, and then run `start.bat` to launch the tool.**
>
> * **Or you can run the `.exe` file for quicker usage.**

## How to use?
**1. Run the `start.bat` file or the `.exe` file.**

**2. Enter your user token, source guild ID and target guild ID, then press enter.**

**3. Type emoji or sticker in the selection field.**

**4. Enter the numbers of the emojis or stickers you want to copy (e.g., 1,7,3,10 or 20-34 or 2,3,45,21-35).**

**5. Press Enter, and the tool will automatically start copying.**

**6. After the tool finishes copying, you can press Enter to return to the menu.**

> [!NOTE] 
> * You must have the `Create Expressions` or `Manage Expressions` permission in the target server.`
